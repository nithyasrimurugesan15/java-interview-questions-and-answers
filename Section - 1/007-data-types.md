# 🔰 Data Types — Interview Questions and Answers

---

## 1️⃣ **What are data types in Java?**

**A:** Data types in Java define the type of data that a variable can store, such as numbers, characters, or boolean values.

---

## 2️⃣ **Why Java is statically typed?**

**A:** Java is statically typed because the data type of a variable is checked at compile time.

---

## 3️⃣ **What are the types of data types in Java?**

**A:** Java has two types of data types: primitive data types and reference data types.

---

## 4️⃣ **List all primitive data types in Java.**

**A:** byte, short, int, long, float, double, char, boolean.

---

## 5️⃣ **Difference between primitive and reference data types?**

**A:** Primitive data types store actual values, while reference data types store the address of objects.

---

## 6️⃣ **Where are primitive data types stored?**

**A:** Primitive data types are typically stored in stack memory.

---

## 7️⃣ **What are default values of data types?**

**A:** Default values are assigned to instance variables when not initialized, such as 0 for int, false for boolean, and null for reference types.

---

## 8️⃣ **Why use long instead of int?**

**A:** long is used when we need to store larger integer values beyond the range of int.

---

## 9️⃣ **Difference between float and double?**

**A:** float is 32-bit with less precision, while double is 64-bit with higher precision.

---

## 🔟 **Why String is not a primitive data type?**

**A:** String is not primitive because it is a class in Java and provides methods for string manipulation.

---

## 1️⃣1️⃣ **What is an object?**

**A:** An object is an instance of a class that contains data and behavior.

---

## 1️⃣2️⃣ **What is an array?**

**A:** An array is a collection of elements of the same data type stored in contiguous memory locations.

---

## 1️⃣3️⃣ **Why are primitive data types faster?**

**A:** Primitive data types are faster because they store values directly in memory without object overhead.

---

## 1️⃣4️⃣ **Can boolean size vary in Java?**

**A:** Yes, the size of boolean is not fixed and depends on JVM implementation.

---

## 1️⃣5️⃣ **When should we avoid using primitive data types?**

**A:** We should avoid primitives when we need null values or object behavior, such as in collections.

---
