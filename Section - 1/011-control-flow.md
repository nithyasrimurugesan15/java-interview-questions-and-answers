# 🔰 Conditional Statements — Interview Questions and Answers

---

## 1️⃣ **What are conditional statements in Java?**

**A:** Conditional statements are used to execute different blocks of code based on specific conditions.

---

## 2️⃣ **What is an if statement?**

**A:** An if statement executes a block of code only when a specified condition is true.

---

## 3️⃣ **Difference between if and if-else?**

**A:** if executes code when condition is true, while if-else provides an alternative block when the condition is false.

---

## 4️⃣ **What is if-else-if ladder?**

**A:** It is used to check multiple conditions sequentially and execute the block of the first true condition.

---

## 5️⃣ **What is nested if statement?**

**A:** A nested if is an if statement inside another if statement.

---

## 6️⃣ **What is switch statement?**

**A:** Switch statement is used to select one block of code from multiple options based on a variable value.

---

## 7️⃣ **Why switch is faster than if-else?**

**A:** Switch can be faster because it may use a jump table for direct execution instead of checking each condition sequentially.

---

## 8️⃣ **What happens if break is not used in switch?**

**A:** Without break, execution continues to the next case (fall-through behavior).

---

## 9️⃣ **Which data types are allowed in switch?**

**A:** int, char, String, enum, byte, short.

---

## 🔟 **What is ternary operator?**

**A:** Ternary operator is a shorthand for if-else: condition ? value1 : value2.

---

## 1️⃣1️⃣ **Difference between switch and if-else?**

**A:** switch is used for fixed values, while if-else is more flexible and can handle ranges and complex conditions.

---

## 1️⃣2️⃣ **Can switch handle ranges?**

**A:** No, switch cannot handle ranges directly.

---

## 1️⃣3️⃣ **Common mistakes in conditional statements?**

**A:** Missing break in switch, using wrong operators, and incorrect condition logic.

---

## 1️⃣4️⃣ **How to reduce nested if complexity?**

**A:** By using switch, early returns, or breaking logic into smaller methods.

---

## 1️⃣5️⃣ **Real-world use of conditional statements?**

**A:** Used in decision-making logic such as validations, authentication, and business rules.

---
