# 🔰 Variables — Interview Questions and Answers

---

## 1️⃣ **What is a variable in Java?**

**A:** A variable is a named memory location used to store data that can be modified during program execution.

---

## 2️⃣ **What are the types of variables in Java?**

**A:** There are three types of variables: local variables, instance variables, and static variables.

---

## 3️⃣ **What is a local variable?**

**A:** A local variable is declared inside a method or block and is accessible only within that method or block.

---

## 4️⃣ **What is an instance variable?**

**A:** An instance variable is declared inside a class but outside methods and is associated with an object.

---

## 5️⃣ **What is a static variable?**

**A:** A static variable is declared with the static keyword and is shared among all instances of a class.

---

## 6️⃣ **Where are variables stored in Java?**

**A:** Local variables are stored in stack memory, while instance and static variables are stored in heap memory.

---

## 7️⃣ **What is variable scope?**

**A:** Variable scope is the region of the program where a variable can be accessed.

---

## 8️⃣ **What is variable lifetime?**

**A:** Variable lifetime is the duration for which a variable exists in memory during program execution.

---

## 9️⃣ **What happens if a variable is not initialized?**

**A:** Local variables cause a compile-time error if not initialized, while instance and static variables get default values.

---

## 🔟 **What is variable shadowing?**

**A:** Variable shadowing occurs when a local variable has the same name as an instance variable, hiding the instance variable.

---

## 1️⃣1️⃣ **What is a final variable?**

**A:** A final variable is a variable whose value cannot be changed once it is assigned.

---

## 1️⃣2️⃣ **Can variables have the same name?**

**A:** Yes, variables can have the same name if they are declared in different scopes.
