# 🔰 Identifiers — Interview Questions and Answers

---

## 1️⃣ **What is an identifier in Java?**

**A:** An identifier is the name given to a variable, method, class, or object to identify it in a Java program.

---

## 2️⃣ **What are the rules for naming identifiers?**

**A:** Identifiers must begin with a letter (a–z, A–Z), underscore (_), or dollar sign ($), cannot start with a digit, cannot be a keyword, and are case-sensitive.

---

## 3️⃣ **Can identifiers start with digits?**

**A:** No, identifiers cannot start with digits.

---

## 4️⃣ **Are identifiers case-sensitive?**

**A:** Yes, Java identifiers are case-sensitive, so "count" and "Count" are different.

---

## 5️⃣ **Can keywords be used as identifiers?**

**A:** No, keywords cannot be used as identifiers because they have predefined meanings in Java.

---

## 6️⃣ **Difference between identifier and variable?**

**A:** Identifier is the name of an entity, while a variable is a memory location used to store data.

---

## 7️⃣ **What is variable shadowing?**

**A:** Variable shadowing occurs when a local variable has the same name as an instance variable, hiding the instance variable within its scope.

---

## 8️⃣ **What are naming conventions for identifiers?**

**A:** Variables and methods use camelCase, classes use PascalCase, and constants use uppercase letters with underscores.

---

## 9️⃣ **Can two identifiers have the same name?**

**A:** Yes, identifiers can have the same name if they are declared in different scopes.

---

## 🔟 **What is the scope of an identifier?**

**A:** Scope is the region of the program where the identifier can be accessed.

---

## 1️⃣1️⃣ **Can identifiers contain special characters?**

**A:** Identifiers can contain underscore (_) and dollar sign ($), but not other special characters.

---

## 1️⃣2️⃣ **Can identifiers be very long?**

**A:** Yes, but they should be meaningful and readable for better code understanding.

---

## 1️⃣3️⃣ **What happens if identifier rules are violated?**

**A:** The Java compiler will throw a compile-time error.

---

## 1️⃣4️⃣ **Can we use Unicode characters in identifiers?**

**A:** Yes, Java allows Unicode characters in identifiers.

---

## 1️⃣5️⃣ **Why are meaningful identifiers important?**

**A:** Meaningful identifiers improve code readability, maintainability, and collaboration.

---
