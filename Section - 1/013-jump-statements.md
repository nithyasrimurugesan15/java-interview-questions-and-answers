# 🔰 Jump Statements — Interview Questions and Answers

---

## 1️⃣ **What are jump statements in Java?**

**A:** Jump statements are used to alter the normal flow of program execution.

---

## 2️⃣ **What is break statement?**

**A:** The break statement is used to terminate a loop or switch statement immediately.

---

## 3️⃣ **What is continue statement?**

**A:** The continue statement skips the current iteration of a loop and moves to the next iteration.

---

## 4️⃣ **What is return statement?**

**A:** The return statement is used to exit from a method and optionally return a value.

---

## 5️⃣ **Difference between break and continue?**

**A:** break exits the loop completely, while continue skips the current iteration and continues with the next one.

---

## 6️⃣ **What is labeled break?**

**A:** Labeled break is used to break out of a specific outer loop in nested loops using a label.

---

## 7️⃣ **Can continue be used in switch?**

**A:** No, continue cannot be used in a switch statement; it is used only in loops.

---

## 8️⃣ **What happens after return statement?**

**A:** The method execution stops immediately, and control returns to the calling method.

---

## 9️⃣ **How does break work in nested loops?**

**A:** break exits only the innermost loop unless used with a label.

---

## 🔟 **When should we use continue?**

**A:** continue is used when we want to skip certain iterations based on a condition.

---

## 1️⃣1️⃣ **What is the performance impact of jump statements?**

**A:** Jump statements have minimal performance impact and are used mainly for control flow.

---

## 1️⃣2️⃣ **Common mistakes with jump statements?**

**A:** Misusing break and continue, leading to incorrect logic or unreachable code.

---

## 1️⃣3️⃣ **Real-world use cases of jump statements?**

**A:** Used in loops for skipping invalid data, exiting loops early, and controlling flow efficiently.

---

## 1️⃣4️⃣ **What is unreachable code?**

**A:** Unreachable code is code that will never be executed, usually written after return or break statements.

---

## 1️⃣5️⃣ **Difference between return and break?**

**A:** return exits the method, while break exits the loop or switch statement.
