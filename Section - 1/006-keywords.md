# 🔰 Keywords — Interview Questions and Answers

---

## 1️⃣ **What are Java keywords?**

**A:** Java keywords are reserved words that have predefined meanings in the language and cannot be used as identifiers.

---

## 2️⃣ **Why keywords cannot be used as identifiers?**

**A:** Keywords cannot be used as identifiers because they are reserved for specific functionalities defined by the Java language.

---

## 3️⃣ **How many keywords are there in Java?**

**A:** Java has around 50+ reserved keywords.

---

## 4️⃣ **Are true, false, and null keywords?**

**A:** No, true, false, and null are not keywords; they are literals.

---

## 5️⃣ **What are the categories of Java keywords?**

**A:** Java keywords can be categorized into data types, access modifiers, control flow statements, exception handling, and class-related keywords.

---

## 6️⃣ **Difference between final, finally, and finalize?**

**A:**  
final → used to declare constants, prevent inheritance, or method overriding  
finally → block used in exception handling that always executes  
finalize → method called by garbage collector before object destruction  

---

## 7️⃣ **What is the static keyword?**

**A:** The static keyword is used to define class-level members that belong to the class rather than instances.

---

## 8️⃣ **What is instanceof keyword?**

**A:** instanceof is used to check whether an object is an instance of a particular class or interface.

---

## 9️⃣ **What are reserved keywords?**

**A:** Reserved keywords are words reserved by Java for future use but not currently used in the language.

---

## 🔟 **Why are keywords important in Java?**

**A:** Keywords define the structure, syntax, and behavior of the Java language, making programs understandable and consistent.

---

## 1️⃣1️⃣ **Can we create variables with names similar to keywords?**

**A:** Yes, but not exactly the same. For example, "className" is allowed, but "class" is not.

---

## 1️⃣2️⃣ **Are all keywords written in lowercase?**

**A:** Yes, all Java keywords are written in lowercase.

---

## 1️⃣3️⃣ **What happens if we use a keyword as an identifier?**

**A:** The compiler will throw a compile-time error.

---

## 1️⃣4️⃣ **Can keywords change in future Java versions?**

**A:** Yes, new keywords can be added in future versions, but existing keywords do not change.

---

## 1️⃣5️⃣ **Give examples of commonly used keywords.**

**A:** class, public, static, void, if, else, for, while, return, new.

---
