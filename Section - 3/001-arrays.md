# 🔰 Arrays — Interview Questions and Answers

---

# 🔹 SECTION 1: ARRAY BASICS

---

## 1️⃣ **What is an array in Java?**

**A:** An array in Java is a collection of elements of the same data type stored in contiguous memory locations.

---

## 2️⃣ **How are arrays stored in memory in Java?**

**A:** Arrays are stored in heap memory, and the reference to the array is stored in stack memory.

---

## 3️⃣ **Difference between primitive array and object array?**

**A:** Primitive arrays store actual values, while object arrays store references to objects.

---

## 4️⃣ **Why array size is fixed in Java?**

**A:** Array size is fixed because memory is allocated at the time of creation and cannot be resized later.

---

## 5️⃣ **What is the default value of array elements?**

**A:** Default values depend on data type: 0 for int, 0.0 for float/double, false for boolean, and null for objects.

---

## 6️⃣ **How do you declare and initialize arrays?**

**A:** Arrays can be declared using `int[] arr` and initialized using `new int[5]` or directly with values `{1,2,3}`.

---

## 7️⃣ **What happens if you access an invalid index?**

**A:** It throws ArrayIndexOutOfBoundsException.

---

## 8️⃣ **Why does Java throw ArrayIndexOutOfBoundsException?**

**A:** To prevent illegal memory access and ensure program safety.

---

## 9️⃣ **How to traverse an array efficiently?**

**A:** Using loops like for-loop or enhanced for-loop.

---

## 🔟 **Difference between length and size()?**

**A:** length is used for arrays, while size() is used for collections.

---

## 1️⃣1️⃣ **Can array size be changed after creation?**

**A:** No, array size is fixed once created.

---

## 1️⃣2️⃣ **What are advantages of arrays?**

**A:** Fast access, memory efficiency, and simple structure.

---

## 1️⃣3️⃣ **What are disadvantages of arrays?**

**A:** Fixed size, lack of flexibility, and no built-in methods.

---

## 1️⃣4️⃣ **What is time complexity of array operations?**

**A:** Access → O(1), Search → O(n), Insert/Delete → O(n).

---

## 1️⃣5️⃣ **Difference between array and ArrayList?**

**A:** Array has fixed size, while ArrayList is dynamic and resizable.
